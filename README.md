# Torrents Api

> API for scraping torrents from 1337x, Piratebay, Eztv, Nyaasi, Torlock and YTS

# How to install

``` bash

# Clone the repo
$ git clone https://github.com/Ryuk-me/Torrents-Api

# Start the server
$ npm start

```

# How it works

```
$ /api/{website name}/{query}/{page(optional)}

```
### Example

```
$ /api/1337x/avengers
```
